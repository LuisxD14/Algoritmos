# Algoritmos
- El archivo inventario.csv contiene algunos productos agregados


Para ejecutar el codigo se debe:
### 1. Clonar el repositorio
```
git clone git@github.com:LuisxD14/Algoritmos.git
cd ~/Algoritmos
```
### 2. Compilar y ejecutar
```
g++ main.cpp manejoArchivo.cpp reporte.cpp crearBuscarId.cpp operacionesArchivo.cpp entradaSalidaP.cpp funcionesAux.cpp -o pan
./pan
```

# Requerimientos Windows
- GCC 14 o superior
- Agregar GCC al PATH
